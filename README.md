# **Интернет магазин по продаже котов Викторов**
### Список библиотек:
* asgiref 3.2.10
* Django 3.1.2
* Pillow 8.0.0
* pytz 2020.1
* sqlparse 0.4.1
### Запуск:
1. `source venv/Scripts/activate`
2. `./manage.py runserver`
3. Открыть http://localhost:8000
### Админка:
* http://localhost:8000/admin
* Логин/пароль - admin
