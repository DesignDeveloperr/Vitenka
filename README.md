# **Интернет магазин по продаже котов Викторов**
## Сайт за час
### Список библиотек:
* Python
  1. asgiref 3.2.10
  2. Django 3.1.2
  3. Pillow 8.0.0
  4. pytz 2020.1
  5. sqlparse 0.4.1
* JavaScript
  1. jQuery
* CSS
  1. TailWind
### Запуск:
1. `docker run -ti --rm -p 8000:8000 docker.pkg.github.com/designdeveloperr/vitenka/vitenka`
2. Открыть http://localhost:8000
### Админка:
* http://localhost:8000/admin
* Логин/пароль - admin
